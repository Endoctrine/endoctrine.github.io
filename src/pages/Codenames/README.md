
# 截码战（Codenames）游戏页面

## 项目概述

这是一个基于 Vue 3 + Tailwind CSS 的截码战桌游网页实现。截码战是一款双人逻辑推理游戏，两队通过隐藏信息和猜测密码来进行对抗。

## 游戏规则

### 游戏进程一：准备阶段
- 每队从词库中随机抽取四个词汇
- 词汇按顺序编号 1~4
- 仅供己方查看（建议记录到纸上防作弊）

### 游戏进程二：加密与截码
1. **加密阶段**：加密人从 [1,2,3,4] 中随机抽取 3 个数字作为密码
2. **描述阶段**：加密人按密码顺序依次描述对应词汇
3. **猜测阶段**：两队分别猜测密码（各猜一次）
4. **公布阶段**：加密人公布真实密码

### 得分规则
- 加密队猜错 → 对手 +1 分
- 截码队猜对 → 加密队再给对手 +1 分

### 胜利条件
率先获得 2 分的队伍获胜。

## 核心功能

### 1. 随机抽词
从预置词库中随机抽取 4 个不重复的词汇。词库包含多个类别：
- 动物类
- 自然类
- 食物类
- 日常物品类
- 抽象概念类
- 动作类
- 地点类
- 情感类
- 颜色类
- 科技类

### 2. 密码生成
从 [1,2,3,4] 中随机抽取 3 个数字并打乱作为密码。

### 3. 防作弊引导
- 抽词后引导玩家将词汇记录到纸上
- 生成密码后引导加密人将密码记录并保密

### 4. 历史记录
记录每轮的密码生成历史，方便回溯。

## 词库自定义

要自定义词库，编辑 `wordBank.js` 文件：

```javascript
export const wordBank = {
  自定义类别: [
    '词汇1',
    '词汇2',
    '词汇3',
    // ... 更多词汇
  ]
};
```

确保每个类别的词汇数量足够多（建议至少 20 个），以保证游戏体验。
